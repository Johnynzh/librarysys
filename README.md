# 广东财经大学图书馆管理系统

# 运行须知

本项目是大学时期的课程设计，执行resourse下面的sql的文件，以及修改application.propeties下的mysql路径，启动项目，访问localhost:8080，账户1，密码1，该账户为管理员账户并且具有权限功能，不同权限的用户其能访问的页面和功能不同，管理员账户具有所有的权限。
